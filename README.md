# 🎯 Bubble Shooter Game

A fun, colorful bubble shooter game perfect for kids and casual gaming!

## 🎮 Play Now
[Play the game here](https://jlmcnamara.github.io/bubble-shooter-game/)

## Features
- 🌈 Colorful bubbles and smooth animations
- 📱 Mobile-friendly touch controls
- 🎯 Progressive difficulty levels
- ✨ Particle effects on bubble pops
- 🏆 Score tracking system

## How to Play
- Click or tap to shoot bubbles at the targets
- Hit targets to score points
- Complete levels by clearing all targets
- Watch out for moving targets!

## Technologies
- Pure HTML5, CSS3, and JavaScript
- Canvas API for smooth graphics
- Progressive Web App ready
